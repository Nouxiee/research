Java Programming Syntaxes

Hello World Example

Function Explanation

Hello World Example public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } } Purpose: Prints "Hello, World!" to the console. Explanation: public class HelloWorld: Defines a class named HelloWorld. public static void main(String[] args): Entry point for the program. System.out.println: Outputs text to the console.

Tagalog Explanation

Hello World Example public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } } Layunin: Ipinapakita ang "Hello, World!" sa console.
Paliwanag:
  • public class HelloWorld: Gumagawa ng isang klase na pinangalanang HelloWorld.
  • public static void main(String[] args): Ang entry point ng programa.
  • System.out.println: Ginagamit upang magpakita ng output sa console.
    • 
// // This is a simple Java program
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

    Basic Data Types

    Function Explanation

    Basic Data Types int number = 10; char letter = 'A'; boolean flag = true; String message = "Hello, Java!";

    Purpose: Demonstrates basic data types in Java. Explanation: int number = 10;: Declares an integer variable and assigns the value 10. char letter = 'A';: Declares a character variable with the value 'A'. boolean flag = true;: Declares a boolean variable with the value true. String message = "Hello, Java!";: Declares a string variable containing text.

    Tagalog Explanation

    Layunin: Ipinapakita ang mga pangunahing data types sa Java.
    Paliwanag:
  • int number = 10;: Nagdeklara ng isang integer na variable at binigyan ng halaga na 10.
  • char letter = 'A';: Nagdeklara ng isang character na variable na may halagang 'A'.
  • boolean flag = true;: Nagdeklara ng isang boolean na variable na may halagang true.
  • String message = "Hello, Java!";: Nagdeklara ng isang string na variable na naglalaman ng text.
    • 
int number = 10;
char letter = 'A';
boolean flag = true;
String message = "Hello, Java!";

    Control Statements

    Function Explanation

    If-Else Statement:

    if (number > 0) { System.out.println("Positive number"); } else { System.out.println("Negative number"); }
    Purpose: Checks if number is positive or negative.

    Explanation:
    If number > 0, the program prints "Positive number."
    Otherwise, it prints "Negative number."

    Tagalog Explanation

    Layunin: Sinusuri kung ang number ay positibo o negatibo.
    Paliwanag: Kapag ang number ay mas malaki sa 0, ipapakita nito ang "Positive number." Kung hindi, ipapakita ang "Negative number."

    For Loop:

    for (int i = 0; i < 5; i++) { System.out.println(i); }
    Purpose: Prints numbers from 0 to 4.

    Explanation:
    int i = 0;: Inilagay ang halaga ng i sa 0. i < 5;: Tinukoy ang kondisyon ng loop. i++: Pinataas ang halaga ng i sa bawat pag-ikot.

    Tagalog Explanation

    Layunin: Ipinapakita ang mga numero mula 0 hanggang 4. Paliwanag: int i = 0; Inilagay ang halaga ng i sa 0. i < 5; Tinukoy ang kondisyon ng loop. i++: Pinataas ang halaga ng i sa bawat pag-ikot.
    
// // If-Else Statement
if (number > 0) {
    System.out.println("Positive number");
} else {
    System.out.println("Negative number");
}

// // For Loop
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

    Methods

    Function Explanation

    Methods public static int add(int a, int b) { return a + b; }
    Purpose: Defines a method to add two integers.

    Explanation: public static int add: Declares a method named add that returns an integer. (int a, int b): Accepts two integer parameters. return a + b;: Returns the sum of a and b.

    Tagalog Explanation

    Layunin: Gumagawa ng isang paraan na nagdaragdag ng dalawang integer.

    Paliwanag: public static int add: Nagdedeklara ng isang paraan na pinangalanang add na nagbabalik ng isang integer. (int a, int b): Tinatanggap ang dalawang integer na parameter. return a + b; : Ibinabalik ang kabuuan ng a at b.
    
// // Method Example
public static int add(int a, int b) {
    return a + b;
}

    Java Variables

    • String - stores text, such as "Hello". String values are surrounded by double quotes
    • int - stores integers (whole numbers), without decimals, such as 123 or -123
    • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
    • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
    • boolean - stores values with two states: true or false

    String Example

    Create a variable called name of type String and assign it the value "John". Then we use println() to print the name variable:

    
// // String Example
String name = "Kevin";
System.out.println(name);

    int Example

    Create a variable called myNum of type int and assign it the value 15:

    
// // int Example
int myNum = 15;
System.out.println(myNum);

    You can also declare a variable without assigning the value, and assign the value later:

    
    int myNum;
    myNum = 15;
    System.out.println(myNum);

    Note that if you assign a new value to an existing variable, it will overwrite the previous value:

    Change the value of myNum from 15 to 20:

    
        int myNum = 15;
        myNum = 20; // myNum is now 20
        System.out.println(myNum);

    Final Variable

    If you don't want others (or yourself) to overwrite existing values, use the final keyword (this will declare the variable as "final" or "constant", which means unchangeable and read-only):

    Example

    
            final int myNum = 15;
            myNum = 20;  // will generate an error: cannot assign a value to a final variable/span>

    Other Types

    A demonstration of how to declare variables of other types:

    Example

    
        int myNum  = 5;

        char myLetter  = 'D';
        boolean myBool  = true;
